# SLIDE 2 — Project Title, Description & Tasks Completed

## Project Titles & Descriptions

### Project 1: TrendReef — AI-Powered Trend Analysis & Content Generation Platform
A multi-agent AI platform that automatically collects social media data from multiple sources, performs trend analysis using weighted scoring algorithms, conducts web research, generates content strategies, and produces platform-specific content — all orchestrated through a LangGraph pipeline with multi-provider LLM support (OpenAI, Anthropic, Azure, Ollama). Built with FastAPI, PostgreSQL (pgvector), Celery, and Docker.

### Project 2: Google Maps Lead Generation — n8n Automated Workflow
An end-to-end no-code automation workflow built in n8n that generates business leads by querying the Google Maps Places API across multiple zip codes and business subcategories. The workflow reads input data from Google Sheets, processes API responses, deduplicates results, writes leads to an output sheet, and tracks processing status — all with built-in retry logic and exponential backoff for reliability.
